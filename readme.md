米时典-小程序
===========================

福州方言在线词典。
项目以taro4+vue3+vite为主要框架，UI与web端保持一致以tailwindcss3为主。
预处理器用stylus、状态管理器用pinia

****

| web | https://seedict.com |
|-----|---------------------
| 公众号 | 米时典SeeDict    |      

### 快速开始
-----------
```
安装
```bash
npm i
或
yarn

运行开发环境
npm run dev:weapp
或
yarn dev:weapp
```

****
##　主要src目录
* [api](#api)
  * 接口请求地址以及公用request方法
* [assets](#其他)
  * 项目静态资源如字体和图片等
* [components](#其他)
  * 项目通用组件
* [pages](#其他)
  * 项目页面
* [store](#其他)
  * 状态管理器
* [styles](#其他)
  * 项目样式文件
* [utils](#其他)
  * 项目工具文件

### api
-----------
| 语法                | 效果                |
|-------------------|-------------------|
| `api`             | 接口请求地址            |
| `request`         | 公用request方法           |

### 其他
-----------
其他目录都与web端项目保持一致
